# Python Programs

Welcome to this beginner-friendly collection of small Python programs. This repository is designed for new learners who are just starting with Python. Each folder contains simple examples and explanations you can run and modify.

Topics covered:
- Variables and Data Types
- If / Else Statements
- For and While Loops
- Functions
- Lists, Tuples, Dictionaries
- File Handling
- Basic OOP (Classes and Objects)

How to use
1. Install Python 3 (https://www.python.org/downloads/) if you don't have it.
2. Clone this repository:
   git clone https://github.com/K-dubey-max/python-programs.git
3. Change into a topic folder and run the examples. For example:
   cd variables_and_data_types
   python variables.py

Notes for beginners
- Each example has comments to explain what it does.
- Try changing values and re-run the scripts to see what happens.
- Don't worry about errors — they help you learn. Read the message, fix the issue, and try again.

Happy learning!
